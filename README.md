# **Online Dress Mart**
## Overview
- Online Dress Mart is an e-commerce platform for the seamless buying and selling of dresses, offering a wide range of collections for men, women, and kids.
## Objective
- To gain more customers
- Customers easy purchase
- Satisfy customer needs
## Project Workflow
- It have
  - SignIn Page
  - Login Page
  - Home Page
  - Men's Collection
  - Women's Collection
  - Kid's Collection
  - Liked Dresses
  - Contact Page
- Admin
  - View
  - Add
  - Update
  - Delete
- Customer
  - View
  - Add to Liked
  - Purchase
## Tech
  - React
  - MongoDB(Database)
